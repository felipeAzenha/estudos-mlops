# estudos-mlops
Este é um repositorio para estudos de MLOps.

Já possuo mais de 15 anos atuando em Machine Learning, porém vejo que o mercado evoluiu em termos não só de ferramentas, mas também de tecnologias. O objetivo é iniciar uma grande revisão estruturada para atualização do formato de trabalho e usarei IA + Python + MLOps para realizar esta tarefa.

Início 08fev26
1.0 Iniciei instalando o Pycharm e o Git, em seguida criei uma pasta em um diretório e montei uma estrutura para organizar o projeto:

meu-projeto-mlops/
├── data/               <-- Coloque o CSV do Kaggle aqui
├── notebooks/          <-- Para experimentos rápidos
├── src/                <-- Onde ficará o código "oficial" (scripts .py)
├── .gitignore          <-- Arquivos que o Git deve ignorar
└── requirements.txt    <-- Lista de bibliotecas (pandas, scikit-learn, etc.)

2.0 Realizei ativação do VENV e criei o gitignore, depois no terminal configurei a comunicação com o git e informei usuario e e-mail para que reconhecesse.
